Hello,
This folder contains all the details of the Music Generation and Visualization work. The following folders contain various information:

exe - here you will find a file that, when launched, will start the program (music generator and visualizer) on your computer. Works for Windows x64 or x86 systems

src/wbdcm - the whole Unity project is located here. Just open (using Unity Editor) the folder contained in this folder and you can change the project. All source files are available without Unity installed - you can view them this way (just in file exlorer). UML diagrams are located in the UML subfolder as well.

src/thesis - LaTeX code is located here

src/UML - UML diagrams are located here

text - here is a test version of the work. Two versions of the same file (.pdf and .ps formats) are contained within
